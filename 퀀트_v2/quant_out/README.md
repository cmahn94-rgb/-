# 퀀트 자동 알림 프로그램 (초보자용 안내)

이 프로젝트는 **주식/암호화폐 종목을 자동으로 분석**해서, 조건에 맞는 “매수 신호”가 나오면 **텔레그램으로 리포트를 보내주는** 프로그램이에요.

## 1) 처음 한 번만 하면 되는 준비

- **파이썬 실행**: `main.py`를 실행하면 필요한 파일들이 자동으로 만들어져요.
- **생기는 파일들**
  - `.env`: 텔레그램 봇 토큰/채팅 ID를 넣는 곳(비밀 정보)
  - `settings.txt`: RSI 기준, 목표 수익률 같은 “전략 설정”
  - `stocks.txt`: 분석할 종목 목록
  - `portfolio.txt`: 내가 보유한 종목(알림용)

## 2) 텔레그램 연결(중요)

처음 실행하고 나면 `.env`가 생기는데, 아래처럼 “여기에_...” 부분을 **진짜 값으로 바꿔야** 텔레그램 전송이 돼요.

예시(형식만 참고):

```
TELEGRAM_TOKEN=123456:ABC-DEF...
TELEGRAM_CHAT_ID=123456789
```

## 2-1) AI 분석(Gemini) 연결(선택)

AI 시장 해석/종목 이유 한 줄을 쓰고 싶으면 `.env`에 아래를 추가로 채워주세요.

```
GEMINI_API_KEY=여기에_Gemini_API_KEY
```

이 값이 없으면 **AI 부분만 자동으로 생략**되고, 나머지 분석/텔레그램 전송은 그대로 진행돼요.

### GitHub Actions로 실행할 때(Secrets 사용)

GitHub 저장소에서 아래처럼 Secrets를 등록하면, `.env` 없이도 자동으로 동작해요.

- **Settings → Secrets and variables → Actions → New repository secret**
  - `TELEGRAM_TOKEN`
  - `TELEGRAM_CHAT_ID`
  - `GEMINI_API_KEY`

그리고 워크플로우 파일(`.github/workflows/run_analysis.yml`)의 `env:`에 위 Secrets를 연결해두면 됩니다.

## 3) 실행 방법

- **한 번만 분석하고 종료**: `main.py` 실행
  - 실행하면 “분석 1회 실행…” 후 리포트를 텔레그램으로 보내요.

## 4) 자주 헷갈리는 포인트

- **왜 파일이 자동으로 생겨요?**  
  처음 쓰는 사람도 바로 실행할 수 있게 “기본 설정 파일”을 만들어주는 거예요.
- **`.env`는 왜 숨겨야 해요?**  
  비밀번호 같은 “비밀키”가 들어 있어서 GitHub 같은 곳에 올라가면 위험해요.
